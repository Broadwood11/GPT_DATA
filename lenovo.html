1 · What IGEL actually is

A modern, hardened OS for secure cloud workspaces

Start here when customers ask "So... is IGEL a thin client?"

IGEL in one sentence

IGEL is an immutable, centrally managed endpoint OS built to provide secure access to virtual, SaaS, and cloud workspaces—without storing data locally.

  • - Immutable, read-only OS — no configuration drift, no "image chaos."
  • - Preventative Security ModelTM — block breaches by design, not after the fact.
  • - No local data — lost/stolen devices don't expose apps, credentials, or files.
  • - Workspace-agnostic — supports Citrix, Omnissa Horizon, AVD, Windows 365, and SaaS.

Simple frame for customers: Lenovo builds the trusted hardware; IGEL is the trusted software layer that makes it a zero-trust endpoint.

What it is · What it isn't

  • - Is: A secure OS for Lenovo endpoints.
  • - Is: A way to simplify VDI, W365, and SaaS access.
  • - Isn't: A hardware vendor or a competing thin client box.
  • - Isn't: Another heavy Windows image to maintain.

Use this line with customers

"Think of IGEL as the locked-down, cloud-optimized OS that makes your Lenovo devices safe to use anywhere—even if the device walks out the door."

5 · Joint architecture

From BIOS posture to cloud workspace session

Use this with architects, EUC leaders, and SecOps.

Technical alignment

  • - Endpoint layer: Lenovo ThinkCentre, ThinkPad, Neo, and ThinkEdge with factory-installed IGEL OS 12 boot into a secure, read-only environment.
  • - Middleware: IGEL UMS manages policies and personas; App Portal delivers modular updates; Insight Services gives telemetry and compliance visibility.
  • - Workspaces: Built-in clients for Citrix, Omnissa Horizon, AVD, W365 plus enterprise browser support for SaaS-heavy workflows.
  • - Security: ThinkShield and Lenovo BIOS Tools are accessible from IGEL for BIOS governance, password control, and firmware posture.

Together, this forms an end-to-end Preventative Security Architecture anchored in hardware and expressed through an immutable OS.

Operational alignment

  • - Deployment: Imaging performed at Lenovo's LITC—devices ship ready for user login.
  • - Management: Persona-based policies via UMS; Lenovo services for lifecycle & break/fix.
  • - Support: Joint escalation path between Lenovo and IGEL; no vendor ping-pong.
  • - Commercial: TruScale DaaS plus IGEL enable predictable OPEX and sustainability reporting.

Architect line

"BIOS-to-session integrity: Lenovo enforces it in hardware; IGEL enforces it in the OS and access path."
