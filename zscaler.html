Essentials

What IGEL actually is

IGEL is a modern, hardened endpoint OS built on a Preventative Security Model. It removes the local attack surface so every session starts compliant and clean.

  • Immutable by design read only, modular OS that cannot be altered by local users.
  • Dataless by intent no user data, apps, or credentials persist locally.
  • Cloud first by nature connects securely to SaaS, VDI, DaaS, or browser based workspaces.
  • Managed at scale one to many control with Universal Management Suite for updates and policy.

Bottom line IGEL converts any device into a secure, compliant Zero Trust endpoint that can be trusted by Zscaler.

Proof points

  • Signed partitions validated at boot
  • USB, clipboard, and peripheral controls
  • Micro updates to locked down components
  • Fast rollout via UD Pocket, PXE, or thin clients
